## 一、Mac/Win
* 订阅地址：请联系作者获取
### 1. Clash Verge
* 下载地址：https://github.com/clash-verge-rev/clash-verge-rev/releases
* 注意事项：建议选择最新的稳定版本（***Latest***）

* 配置使用 Clash Verge  
  - 复制【订阅地址】  
  - 打开 Clash Verge，【订阅】 → 【粘贴复制的订阅地址】 → 【导入】  
  - 【订阅】 → 【选择已订阅的节点】 → 【右键】 → 【编辑信息】 → 修改【名称】  
  - 如果需要自定义代理规则，【订阅】 → 【选择已订阅的节点】 → 【右键】 → 【编辑规则】  
  - 任务栏图标快捷设置  
    - 出站模式：规则模式
    - 订阅：选择已订阅的名称
    - Proxy：选择代理节点（建议选择美国ChatGPT解锁节点）
    - 系统代理：勾选

## 二、iOS/Android
* 订阅地址：请联系作者获取
### 1. iOS：Shadowrocket（App Store 美区）
* 复制【订阅地址】
* 右上角+号 → 【Type 选择 Subcribe】 → 确认【URL】 → 【Save】
* 【Connectivity Test】 → 点击最右侧的【T】修改为【CONNECT】
* 点击【Connectivity Test】 测试节点的连通性，等待出结果
* 下滑选择可用的美国代理 → 点击【Not Connected】使用代理

### 2. Android：NekoBox
* 下载地址：https://github.com/Matsuridayo/NekoBoxForAndroid/releases
* 复制【订阅地址】
* 【Configurationg】 → 点击右上角【+号】 → 【Import from Clipboard】
* 【Group】 → 修改添加的订阅 → 修改【Group name】 → 下滑找到打开【Auto Update】 → 点击右上角保存
* 【Route】 → 开启【Domain rule for 中国】和【IP rule for 中国】
* 【Configurationg】 → 点击右上角【三个点】 → 【URL Test】，等待出结果
* 【Configurationg】 → 下滑选择可用的美国代理 → 点击界面底部的纸飞机图标使用代理
