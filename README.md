## 三、配置使用 Clash Verge  
1. 复制【订阅地址】  
2. 打开 Clash Verge，【订阅】 → 【粘贴复制的订阅地址】 → 【导入】  
3. 【订阅】 → 【选择已订阅的节点】 → 【右键】 → 【编辑信息】 → 修改【名称】  
4. 如果需要自定义代理规则，【订阅】 → 【选择已订阅的节点】 → 【右键】 → 【编辑规则】  
5. 任务栏图标快捷设置  
* 出站模式：规则模式
* 订阅：选择已订阅的名称
* Proxy：选择代理节点（建议选择美国ChatGPT解锁节点）
* 系统代理：勾选
